# Voidwear

A modern web application featuring a React/Vite frontend and a Node.js/Express backend.

## Project Structure

- `Frontend/`: The client-side application built with React, Vite, Redux Toolkit, and Tailwind CSS.
- `Backend/`: The server-side API built with Node.js and Express.

## Features

- Modern UI with Tailwind CSS
- State management with Redux Toolkit
- Authentication system
- Clean and modular project structure

## Setup Instructions

### Backend
1. Navigate to the `Backend` directory.
2. Run `npm install` to install dependencies.
3. Start the server with `npm start` or `node server.js`.

### Frontend
1. Navigate to the `Frontend` directory.
2. Run `npm install` to install dependencies.
3. Start the development server with `npm run dev`.

## Technologies Used

- **Frontend**: React, Vite, Redux, Axios, Tailwind CSS
- **Backend**: Node.js, Express
- **State Management**: Redux Toolkit
- **API**: Axios
